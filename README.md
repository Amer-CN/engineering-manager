# 工程管家 - 工程业务管理系统

## 📖 项目介绍

**工程管家** 是一款专为工程公司和施工团队设计的桌面应用程序，帮助您高效管理项目、合同、发票、人员、材料、结算等核心业务。

## ✨ 功能一览

- 📊 **仪表盘** - 项目总览、合同统计、发票提醒、任务待办
- 🏗️ **项目管理** - 创建项目、健康度评分、收支对比、成本分析
- 📝 **合同管理** - 收入合同 / 支出合同、结算金额、收款统计
- 🧾 **发票管理** - 开票登记、收款联动、状态跟踪（已开具/部分收款/已收票/作废/红冲）
- 🚜 **结算管理** - 结算单创建、审批流程、付款记录
- 📄 **模板管理** - Word/Excel 模板上传、变量填充生成、7 种分类（合同/结算/用印/用款/红头/函件/其他）
- 👷 **员工管理** - 班组管理、人员流动、入场/离场记录
- 📦 **仓库管理** - 物料库、出入库记录、项目材料台账
- 🏢 **单位管理** - 合作单位、监管单位、纳税资质、联网查询
- 💰 **工资管理** - 考勤打卡（画笔日历+逐日标记）、工资计算（日薪/月薪制+社保公积金）、工资统计
- 📊 **成本台账** - 真实资金流追踪（经营支出/垫资/股东融资），双入口角色分离（财务侧边栏录入+领导项目Tab分析看板），凭证号按项目自增，Excel级列筛选
- 👤 **用户与权限** - 头像弹出菜单（用户管理/系统设置/锁定屏幕）、角色权限矩阵编辑、操作日志审计
- 🔒 **屏幕锁定** - 一键锁屏防窥屏，需密码解锁后继续操作

## 🛠️ 技术栈

- **Electron 28** - 跨平台桌面应用框架
- **React 18 + TypeScript 5** - 类型安全的 UI 开发
- **Vite 5** - 极速构建工具
- **TailwindCSS** - 实用优先的样式框架
- **framer-motion** - 全站交互动画引擎（页面过渡/组件微交互/数据可视化动画）
- **recharts** - 声明式数据可视化图表库（BarChart/PieChart/RadialBarChart）
- **lucide-react** - SVG 图标库（tree-shakeable，60+ 图标）
- **JSON 文件存储** - 本地数据持久化，无需数据库
- **Tesseract.js** - 离线 OCR 识别

## 🚀 运行项目

### 环境要求

- Node.js 18+
- Windows 系统（推荐）

### 安装依赖

```bash
npm install
```

### 开发模式运行

双击运行 `工程管家.bat`，或手动执行：

```bash
npm run dev
```

### 打包桌面应用

```bash
npm run build
```

打包完成后，安装包位于 `release/` 目录。

## 📝 使用说明

1. 首次使用需要先创建项目
2. 在项目中可以添加任务、人员、材料等
3. 所有数据保存在本地，无需担心数据安全
4. 支持 Windows 和 Mac 系统
