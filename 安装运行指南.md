# 工程管家 - 安装运行指南

## 📋 准备工作

### 第一步：安装 Node.js（必需）

1. 访问 Node.js 官网下载安装包
   - 官网地址：https://nodejs.org/
   - 下载推荐版本（LTS 版本）

2. 安装过程：
   - 双击下载的安装包
   - 点击 "Next" 继续
   - 同意许可协议
   - 选择安装路径（默认即可）
   - 点击 "Install" 安装
   - 点击 "Finish" 完成

3. 验证安装：
   - 打开 PowerShell 或 CMD
   - 输入以下命令验证：
   ```powershell
   node --version
   npm --version
   ```
   - 如果显示版本号（如 v18.x.x 和 9.x.x），说明安装成功

### 第二步：打开项目

1. 进入项目目录
   ```powershell
   cd "e:\测试"
   ```

2. 或者在文件资源管理器中：
   - 打开 "此电脑"
   - 进入 e:\测试 文件夹
   - 在地址栏输入 `cmd` 并回车，打开命令窗口

## 🚀 运行项目

### 安装依赖

在命令窗口中执行：

```powershell
npm install
```

> 注意：首次安装可能需要 5-10 分钟，请耐心等待

### 开发模式运行

安装完成后，执行：

```powershell
npm run dev
```

等待几秒钟，会自动打开应用窗口。

## 📦 打包桌面应用

### 执行打包命令

```powershell
npm run build
```

或使用更详细的命令：

```powershell
npm run electron:build
```

### 打包产物

打包完成后，安装包会在 `release` 文件夹中：

| 文件 | 说明 |
|------|------|
| `release/工程管家-Setup-1.0.0.exe` | Windows安装包（推荐） |
| `release/win-unpacked/` | 便携版（无需安装） |
| `release/mac/` | macOS版本 |
| `release/linux/` | Linux版本 |

### 便携版使用方法

1. 进入 `release/win-unpacked/` 目录
2. 双击 `工程管家.exe` 即可运行
3. 数据存储在程序同目录下的 `userData` 文件夹中

## 🔐 百度OCR配置

### 为什么需要配置？

| 模式 | 识别内容 | 所需条件 |
|------|---------|---------|
| 离线模式 | 仅身份证号 | 无需配置，直接可用 |
| 百度OCR | 全部信息 | 需要百度API密钥 |

### 配置步骤

1. 打开 **工程管家** 应用
2. 进入 **系统设置** 页面
3. 找到 **OCR文字识别设置**
4. 选择 **百度OCR** 模式
5. 输入从百度智能云获取的 API Key 和 Secret Key
6. 点击 **保存配置**

### 如何获取百度API？

**详细指南请查看**: [docs/百度OCR配置指南.md](docs/百度OCR配置指南.md)

**简要步骤：**
1. 访问 https://console.bce.baidu.com/
2. 搜索"身份证识别"或进入"文字识别服务"
3. 开通服务（个人用户有免费额度）
4. 创建应用，获取 API Key 和 Secret Key

## 📁 数据存储

软件数据默认存储在用户目录：

```
C:\Users\[用户名]\AppData\Roaming\engineering-manager\
├── engineering.json  (业务数据)
├── config.json       (配置信息)
└── uploads\          (上传的文件)
```

### 修改数据存储位置

1. 进入 **系统设置** 页面
2. 点击 **更改数据存储位置**
3. 选择新的存储目录
4. 数据会自动迁移到新位置

**提示**：建议将数据存储在非系统盘（如D盘），便于重装系统后恢复数据。

## 📖 快速使用教程

### 1. 创建第一个项目
- 打开应用后，点击左侧菜单的 "项目管理"
- 点击右上角 "新建项目" 按钮
- 填写项目名称、地址、预算等信息
- 点击 "创建" 完成

### 2. 添加团队成员
- 点击左侧 "人员管理"
- 点击 "添加成员"
- 填写姓名、职位、联系方式
- 点击 "添加" 完成

### 3. 创建任务
- 点击左侧 "任务管理"
- 点击 "新建任务"
- 选择所属项目、负责人
- 设置优先级和截止日期
- 点击 "创建" 完成

### 4. 管理材料
- 点击左侧 "材料管理"
- 点击 "添加材料"
- 填写材料名称、数量、单价
- 点击 "添加" 完成

### 5. 记录费用
- 点击左侧 "成本管理"
- 点击 "记录费用"
- 选择项目、填写金额
- 点击 "记录" 完成

### 6. 上传图纸
- 点击左侧 "图纸管理"
- 点击 "上传图纸"
- 选择项目、选择文件
- 点击 "上传" 完成

## ❓ 常见问题

### Q1: npm install 失败？
**解决方案：**
- 检查网络连接
- 清理 npm 缓存：`npm cache clean --force`
- 删除 node_modules 文件夹后重试

### Q2: 运行报错 "electron" 找不到？
**解决方案：**
- 删除 node_modules 和 package-lock.json
- 重新执行 `npm install`

### Q3: 打包失败？
**解决方案：**
- 确保有足够的磁盘空间
- 以管理员身份运行命令窗口
- 检查杀毒软件是否拦截

### Q4: 界面显示异常？
**解决方案：**
- 按 Ctrl+Shift+I 打开开发者工具
- 查看控制台是否有错误信息
- 重启应用

### Q5: 百度OCR识别失败？
**解决方案：**
- 确认网络连接正常
- 检查百度API配置是否正确
- 可切换到离线模式使用

### Q6: 如何打开开发者工具？
**解决方案：**
- 按 `Ctrl+Shift+I` 可打开控制台
- 查看日志和调试信息
