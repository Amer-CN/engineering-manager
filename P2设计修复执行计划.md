## 第 7 步：统一按钮样式（规划中）

### 目标
将所有不规范的按钮用法统一为 `btn-*` 组件类，确保按钮样式一致性。

### 当前问题
扫描发现 **96 处**不规范的按钮用法，主要问题：
1. **直接写 primary 颜色**：`bg-primary-600 hover:bg-primary-700 text-white rounded-lg`
2. **错误颜色**：`hover:bg-blue-700`（应为 `hover:bg-primary-700`）
3. **取消按钮不一致**：`text-slate-700 hover:bg-slate-100 rounded-lg`
4. **小按钮不一致**：`px-2 py-1 text-xs ...`

### 修复映射表

| 不规范用法 | 规范用法 | 适用场景 |
|-----------|----------|----------|
| `bg-primary-600 hover:bg-primary-700 text-white rounded-lg` | `btn btn-primary` | 主要操作按钮 |
| `bg-primary-600 hover:bg-blue-700 ...` （错误！）| `btn btn-primary` | 主要操作按钮（需修正颜色） |
| `text-slate-700 hover:bg-slate-100 rounded-lg` | `btn btn-secondary` | 取消/关闭按钮 |
| `border border-slate-300 rounded hover:bg-slate-50` | `btn btn-secondary` | 分页/辅助按钮 |
| `text-red-600 hover:bg-red-50 rounded` | `btn btn-danger btn-sm` | 删除/危险操作 |
| `text-primary-600 hover:bg-primary-50 rounded` | `btn btn-ghost btn-sm` | 编辑/详情链接按钮 |
| `bg-indigo-600 hover:bg-indigo-700 text-white rounded-lg` | `btn btn-primary` | HR模块按钮（颜色需统一） |
| `bg-amber-600 hover:bg-amber-700 text-white rounded-lg` | `btn btn-warning btn-sm` | 薪资历史按钮 |

### 可用按钮类（`src/index.css`）

| 类 | 用途 | 样式 |
|----|------|------|
| `btn` | 基础按钮 | `px-4 py-2 rounded-lg font-medium` |
| `btn-primary` | 主要按钮 | `bg-primary-600 text-white hover:bg-primary-700` |
| `btn-secondary` | 次要按钮 | `bg-white border border-slate-200 text-slate-700 hover:bg-slate-50` |
| `btn-success` | 成功按钮 | `bg-success-600 text-white hover:bg-success-700` |
| `btn-danger` | 危险按钮 | `bg-danger-500 text-white hover:bg-danger-600` |
| `btn-warning` | 警告按钮 | `bg-warning-500 text-white hover:bg-warning-600` |
| `btn-ghost` | 幽灵按钮 | `bg-transparent text-slate-600 hover:bg-slate-100` |
| `btn-sm` | 小按钮 | `px-3 py-1.5 text-sm` |
| `btn-lg` | 大按钮 | `px-6 py-3 text-lg` |
