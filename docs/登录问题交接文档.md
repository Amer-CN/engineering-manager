## 三、登录流程分析

```
用户点击登录
    ↓
Login.tsx handleSubmit()
    ↓
window.electronAPI.login(username, password)
    ↓ (IPC)
electron/ipc-handlers/auth.ts
    ↓
verifyPassword() 验证密码
    ↓
返回用户数据 { success: true, data: {...} }
    ↓
前端 login(userData) 保存到 Context
    ↓
localStorage.setItem('engineering_auth', ...)
    ↓
window.location.reload()
    ↓
AuthContext 初始化时从 localStorage 恢复登录状态
    ↓
App.tsx 判断 isAuthenticated = true，显示主界面
```

## 五、可能的问题原因

### 1. preload.js 未正确加载

**症状**: `window.electronAPI` 是 undefined

**检查**: 打开 DevTools，输入 `console.log(window.electronAPI)`

**解决方案**: 检查主进程的 `webPreferences.preload` 配置

### 2. 表单提交被阻止

**症状**: 没有任何控制台输出，按钮点击无反应

**检查**: form 的 onSubmit 是否正确绑定

**解决方案**: 添加 `e.preventDefault()` 和调试日志

### 3. 数据库初始化失败

**症状**: 无法找到用户 `admin`

**检查**: engineering.json 文件是否存在，数据路径是否正确

### 4. 跨域/安全策略问题

**症状**: IPC 调用失败

**检查**: BrowserWindow 的 `contextIsolation` 和 `nodeIntegration` 配置
