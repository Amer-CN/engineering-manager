# 百度OCR配置指南

## 概述

本软件支持百度OCR在线识别功能，可以自动识别身份证上的姓名、身份证号、性别、民族、地址、出生日期等信息。

## 快速配置

### 方法一：在软件内配置

1. 打开 **工程管家** 软件
2. 进入 **系统设置** 页面
3. 在 **OCR识别设置** 中选择 **百度OCR** 模式
4. 输入您的 API Key 和 Secret Key
5. 点击 **保存配置**

### 方法二：导入配置文件

如果您已有配置文件，可以手动添加到浏览器的 LocalStorage 中。

## 申请百度OCR API

### 步骤 1：注册百度账号

访问 [百度智能云控制台](https://console.bce.baidu.com/)，注册并登录账号。

### 步骤 2：开通身份证识别服务

1. 登录后进入控制台
2. 搜索「身份证识别」或进入「文字识别服务」
3. 点击「开通服务」
4. 个人用户可免费领取资源包

### 步骤 3：创建应用

1. 进入「应用列表」
2. 点击「创建应用」
3. 填写应用信息：
   - 应用名称：`工程管家`（或其他名称）
   - 应用类型：客户端
   - 功能描述：身份证识别
4. 创建成功后，获取 **API Key** 和 **Secret Key**

### 步骤 4：配置到软件

将获取的 API Key 和 Secret Key 填入软件的 OCR 设置中。

## 离线模式

如果您不配置百度OCR，软件会使用 **离线模式**，仅能识别身份证号，其他信息需要手动输入。

## 常见问题

### Q: 识别失败怎么办？
A: 检查网络连接，确保可以访问 `aip.baidubce.com`

### Q: API Key 泄露了怎么办？
A: 立即在百度智能云控制台删除该应用，重新创建新的 API

### Q: 免费额度用完了怎么办？
A: 购买资源包，或等待次日零点重置
